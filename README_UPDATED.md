## 🚀 البدء السريع

### المتطلبات الأساسية
- Node.js 16+
- npm أو yarn
- قاعدة بيانات PostgreSQL

### الخطوات

1. **تشغيل الخادم الخلفي** (في Terminal جديد)
```bash
cd server
npm start
```

2. **تشغيل الموقع** (في Terminal آخر)
```bash
npm run dev
```

3. **فتح المتصفح**
```
http://localhost:5173
```

## 🔧 التخصيص والتطوير

### إضافة ترجمة جديدة

```typescript
// في src/lib/i18n.ts
export const translations = {
  en: { /* ... */ },
  ar: { /* ... */ },
  fr: {
    nav: { signin: 'Se connecter', /* ... */ },
    landing: { /* ... */ }
  }
};

// حدث النوع
export type Language = 'en' | 'ar' | 'fr';
```

### استخدام الترجمة في مكونات جديدة

```tsx
import { useLanguage } from '../context/LanguageContext';
import { useTranslation } from '../../lib/i18n';

export function MyComponent() {
  const { language } = useLanguage();
  const t = useTranslation(language);

  return <h1>{t.landing.subtitle}</h1>;
}
```
