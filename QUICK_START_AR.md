# 🚀 دليل البدء السريع

## الخطوات الثلاث لتشغيل الموقع

### ✅ الخطوة 1: فتح Terminal جديد وشغّل الخادم
```bash
cd server
npm start
```
**الحالة المتوقعة:** `🚀 Server running in development mode`

### ✅ الخطوة 2: فتح Terminal ثاني وشغّل الموقع
```bash
npm run dev
```
**الحالة المتوقعة:** `➜  Local:   http://localhost:5173/`

### ✅ الخطوة 3: فتح المتصفح
اذهب إلى: **http://localhost:5173**

---

## ✨ اختبر المميزات الجديدة

### 🌍 تبديل اللغة
1. انقر على أزرار اللغة في الأعلى اليمين
2. اختر "EN" للإنجليزية
3. اختر "العربية" للعربية
4. لاحظ التغيير التلقائي للـ RTL

### 🎨 البطاقات 3D
1. انزل للأسفل
2. شاهد 4 بطاقات ملونة تمثل المميزات
3. مرّر الماوس فوق البطاقات
4. لاحظ الحركات السلسة والتأثيرات الجميلة

## ⚠️ ملاحظات مهمة

- **كلا الـ Terminals يجب أن يعملا** في نفس الوقت
- **الخادم على المنفذ 5000**, الموقع على **5173**
- **لا تغلق أي من الـ Terminals** أثناء التطوير

---

## 🐛 حل المشاكل

| المشكلة | الحل |
|--------|------|
| الموقع لا يحمل | تأكد من تشغيل الخادم (Terminal 1) |
| الأزرار لا تعمل | افسح الـ cache (Ctrl+Shift+Delete) |
| لا تظهر البطاقات | افسح الـ cache وأعد تحميل (Ctrl+R) |
| خطأ في الترجمة | تأكد من تثبيت جميع المتطلبات |
