## 🚀 كيفية الاستخدام

### تشغيل السكريبت

```bash
cd server
node prisma/seed-pet-images.js
```

### ماذا يفعل السكريبت؟

1. **تحديث الحيوانات الموجودة**
   - يجلب جميع الحيوانات من قاعدة البيانات
   - يضيف 1-3 صور عشوائية لكل حيوان حسب نوعه

2. **إنشاء حيوانات جديدة**
   - ينشئ 30 حيوان جديد
   - كل حيوان يحتوي على:
     - اسم فريد
     - معلومات كاملة (السلالة، العمر، الوزن، اللون)
     - 1-4 صور احترافية
     - حالة التطعيم والشهادات
     - السعر ومعلومات التزاوج

## 🔄 إعادة تشغيل السكريبت

يمكنك تشغيل السكريبت عدة مرات:
- **المرة الأولى**: يضيف صور للحيوانات الموجودة + ينشئ 30 حيوان جديد
- **المرات التالية**: يحدّث الصور فقط + ينشئ 30 حيوان جديد في كل مرة

```bash
# تشغيل مرة أخرى لإضافة المزيد من الحيوانات
node prisma/seed-pet-images.js
```

## 🛠️ استكشاف الأخطاء

### الخطأ: "Cannot read properties of undefined"
**الحل:** تأكد من:
- تشغيل الخادم
- الاتصال بقاعدة البيانات
- تثبيت Prisma Client: `npx prisma generate`

### الخطأ: "User not found"
**الحل:** قم بتشغيل seed الأساسي أولاً:
```bash
node prisma/seed.js
```

### الصور لا تظهر
**الحل:**
- تحقق من اتصال الإنترنت
- تأكد من عمل Unsplash
- جرّب تحديث الصفحة (Ctrl+F5)
