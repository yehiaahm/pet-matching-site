## 🚀 كيفية الاستخدام

### للمستخدم:

```
1. افتح الموقع
2. انزل لقسم FAQ (الأسئلة الشائعة)
3. اضغط "📧 اتصل بنا"
4. ملء النموذج:
   • الاسم
   • البريد الإلكتروني
   • الفئة (تقنية، حساب، دفع، إلخ)
   • الموضوع

6. رسالة نجاح تظهر فوراً ✅
```

```
1. سجل دخول كـ ADMIN

3. اضغط عليه
4. تظهر لوحة التحكم الكاملة
5. اختر شكوى من القائمة
6. اقرأ التفاصيل
7. اكتب الرد في الصندوق
8. اضغط "إرسال"
9. غيّر الحالة عند الانتهاء
10. الشكوى محفوظة تلقائياً ✅
```

## 🧪 الاختبار

### اختبار سريع:

```bash
# 1. إرسال شكوى
curl -X POST http://localhost:3000/api/v1/support/tickets \
  -H "Content-Type: application/json" \
  -d '{
    "name": "أحمد",
    "email": "ahmed@test.com",
    "subject": "مشكلة تقنية",
    "category": "TECHNICAL",
    "message": "لا يعمل الموقع",
    "priority": "HIGH"
  }'

# 2. عرض جميع الشكاوى (كمسؤول)
curl -X GET http://localhost:3000/api/v1/support/admin/tickets \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"

# 3. الرد على شكوى
curl -X POST http://localhost:3000/api/v1/support/tickets/TICKET_ID/reply \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "تم حل المشكلة"
  }'

# 4. إغلاق التذكرة
curl -X PATCH http://localhost:3000/api/v1/support/admin/tickets/TICKET_ID/status \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "CLOSED"
  }'
```
