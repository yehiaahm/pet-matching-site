# تحديثات الموقع - دعم اللغة العربية والإنجليزية وصور 3D

## المميزات الجديدة المضافة:

### 1. نظام تبديل اللغة (Internationalization - i18n)

تم إضافة نظام كامل لدعم لغات متعددة:

**الملفات المضافة:**
- `src/lib/i18n.ts` - ملف الترجمة الرئيسي يحتوي على جميع النصوص باللغة الإنجليزية والعربية
- `src/app/context/LanguageContext.tsx` - Context API لإدارة حالة اللغة على مستوى التطبيق
- `src/app/components/LanguageSwitcher.tsx` - زر تبديل اللغة في الرأس

**المميزات:**
- تبديل فوري بين العربية والإنجليزية
- حفظ تفضيل اللغة في `localStorage`
- تغيير اتجاه الصفحة تلقائياً (RTL للعربية، LTR للإنجليزية)
- جميع نصوص الصفحة الرئيسية مترجمة

### 2. صور 3D تفاعلية للمميزات

تم إضافة قسم جديد بعنوان "Features3D" يعرض مميزات الموقع بطريقة عصرية:

**الملفات المضافة:**
- `src/app/components/Features3D.tsx` - مكون يعرض 4 بطاقات 3D متحركة

**المميزات:**
- 4 بطاقات ثلاثية الأبعاد تعرض:
  1. **مطابقة ذكية بـ AI** - خوارزمية التطابق الذكية
  2. **البحث بالموقع GPS** - البحث الجغرافي
  3. **السجلات الصحية** - التوثيق الصحي
  4. **دعم المجتمع** - التواصل مع المربيين

- تأثيرات حركية عند التمرير (hover)
- تأثيرات دخول متسلسلة
- إحصائيات ديناميكية (معدل النجاح، الدعم 24/7، الأمان)
- دعم اللغة العربية والإنجليزية

### 3. تحديثات الصفحة الرئيسية (Landing Page)

**التحديثات:**
- إضافة زر تبديل اللغة في الرأس
- تبديل جميع النصوص لاستخدام نظام الترجمة
- إضافة قسم Features3D بعد الميزات التقليدية
- دعم RTL/LTR ديناميكي بناءً على اللغة المختارة

### 4. التكامل مع التطبيق الرئيسي (App.tsx)

**التحديثات:**
- إضافة `LanguageProvider` كـ wrapper لجميع routes
- توفر خيار تغيير اللغة في كل أنحاء التطبيق

---

## كيفية الاستخدام:

### لتبديل اللغة:
1. انقر على زر اللغة في الرأس ("EN" أو "العربية")
2. سيتم حفظ اختيارك تلقائياً

### لإضافة ترجمات جديدة:
1. عدّل ملف `src/lib/i18n.ts`
2. أضف النصوص الجديدة للعربية والإنجليزية
3. استخدمها في المكونات عبر `useLanguage()` و `useTranslation()`

### لإضافة مكونات 3D جديدة:
1. انسخ نمط `Feature3DCard` من `Features3D.tsx`
2. عدّل الـ icon والـ gradient والنصوص
3. أضفها إلى مصفوفة `features`
