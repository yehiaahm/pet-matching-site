# 🐕 موقع مطابقة الحيوانات الأليفة - PetMat

## ✅ كل شيء متصل والبيانات تُحفظ بشكل صحيح!

### 🎯 ما يجب أن تعرفه

هذا الموقع مبني بشكل احترافي حيث:
- ✅ **Frontend** متصل بـ **Backend** بشكل آمن
- ✅ **Backend** متصل بـ **PostgreSQL** عبر Prisma
- ✅ **جميع البيانات** تُحفظ تلقائياً في قاعدة البيانات
- ✅ **نظام API موحد** لسهولة التطوير والصيانة

## 🔗 كيف تعمل الاتصالات؟

### 1. Frontend يرسل طلب
```typescript
import { petsApi } from '@/lib/api';

// جلب جميع الحيوانات
const pets = await petsApi.getAll();
```

### 2. Backend يستقبل الطلب
```javascript
// server/routes/petRoutes.js
router.get('/', getAllPets);
```

### 3. Backend يجلب من Database
```javascript
// server/controllers/petController.js
const pets = await prisma.pets.findMany();
```

### 4. البيانات تُرسل للـ Frontend
```
Database → Backend → Frontend → العرض في الموقع
```

## 💾 البيانات المخزنة

جميع العمليات التالية تُحفظ في قاعدة البيانات `pets_db`:

| العملية | الجدول | المثال |
|---------|--------|---------|
| تسجيل مستخدم | `users` | البريد، الاسم، كلمة السر |
| إضافة حيوان | `pets` | الاسم، النوع، السلالة |
| طلب تزاوج | `breeding_requests` | الحيوان المبادر، المستهدف |
| إرسال رسالة | `messages` | المرسل، المستقبل، المحتوى |
| تقييم | `reviews` | التقييم، التعليق |

## 🔍 عرض البيانات المخزنة

### الطريقة 1: Prisma Studio (موصى بها)
```bash
cd server
npx prisma studio
```
سيفتح واجهة مرئية على: **http://localhost:5555**

### الطريقة 2: سطر الأوامر
```bash
node -e "
const { PrismaClient } = require('@prisma/client');
const p = new PrismaClient();
(async () => {
  console.log('👥 المستخدمين:', await p.users.count());
  console.log('🐕 الحيوانات:', await p.pets.count());
  console.log('💕 الطلبات:', await p.breeding_requests.count());
  await p.\$disconnect();
})();
"
```

## 📁 الملفات المهمة

### إعدادات الاتصال
- `.env` - إعدادات Frontend (API URL)
- `server/.env` - إعدادات Backend (Database, Port)

### نظام API
- `src/lib/api.ts` - نظام API موحد للـ Frontend

- `server/routes/` - جميع مسارات API
- `server/controllers/` - معالجات الطلبات

npx prisma db push
