# 🎯 خطوات التشغيل الفورية (Windows)

## الخطوة 1: افتح 3 نوافذ في PowerShell

### نافذة 1️⃣: الخادم الخلفي
```powershell
cd server
npm start
```
**ستظهر الرسالة**: `🚀 Server running on http://localhost:5000`

---

## نافذة 2️⃣: خدمة AI

### إعداد Python Environment (مرة واحدة فقط):
```powershell
cd ai-service
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### تشغيل الخدمة:
```powershell
cd ai-service
.\.venv\Scripts\activate
uvicorn main:app --host 0.0.0.0 --port 8001
```
**ستظهر الرسالة**: `Uvicorn running on http://0.0.0.0:8001`

---

## نافذة 3️⃣: Frontend

```powershell
npm run dev
```
**ستظهر الرسالة**: `➜  Local: http://localhost:5173`

## ⚠️ استكشاف المشاكل

| المشكلة | الحل |
|--------|------|
| خطأ "AI service unavailable" | تأكد من تشغيل نافذة 2 (AI) |
| CORS blocked error | أضف origin في `server/.env` |
| Database connection error | تحقق من `DATABASE_URL` في `.env` |
| Socket.io not connecting | تأكد من تسجيل الدخول أولاً |
| npm ERR! | احذف `node_modules` و `npm install` مجدداً |
