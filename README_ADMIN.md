## 🚀 البدء السريع

### 1. Database Setup:
```bash
cd server
npx prisma migrate dev --name add_admin_system
```

### 2. Create Admin:
```sql
UPDATE users SET role = 'SUPER_ADMIN' WHERE email = 'your-email@example.com';
```

### 3. Access:
```
http://localhost:5173/admin
```
